#  Tic Tac Toe - 2 Player Console Game (Java)

A simple **2-player Tic Tac Toe** game built in **Java**, where two users take turns marking the spaces in a 3Ã—3 grid. The game announces the winner or declares a draw when the grid is full.

---

##  Features

- Two-player interactive gameplay  
- Player name input for personalized experience  
- Real-time board updates after each move  
- Valid move checking (prevents overwriting cells)  
- Win and draw detection with clear results  

---

##  How to Play

1. Run the program in any Java-supported environment.  
2. Enter the names of **Player 1 (X)** and **Player 2 (O)**.  
3. Players take turns entering the **row (1-3)** and **column (1-3)** positions.  
4. The first player to align three of their marks (horizontally, vertically, or diagonally) wins.  
5. If all cells are filled without a winner, the game ends in a **draw**.
